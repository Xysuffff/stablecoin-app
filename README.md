# Stablecoin App

Experimental stablecoin dashboard application built with Next.js and modern web technologies.

## Overview

This project explores the concept of a stablecoin management platform with a modern fintech-style interface and scalable frontend architecture.

## Features

- Responsive landing page
- Stablecoin dashboard UI
- Wallet balance simulation
- Transaction activity interface
- Modern fintech-inspired design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React

## Project Structure

```bash
app/        # Application routes and pages
public/     # Static assets
```

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Status

This project is currently in active development and serves as an experimental stablecoin application prototype.
